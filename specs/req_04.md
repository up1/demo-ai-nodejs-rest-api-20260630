## Plugins

### New

### [Indented Table](https://community.obsidian.md/plugins/indented-table)

Renders markdown pipe tables that are indented inside list items.

### [Konbini Kanban](https://community.obsidian.md/plugins/konbini-kanban)

Organize notes as a Linear-style kanban board in a Bases view, with status columns, drag-to-restatus, priority, labels, and sub-issues.

### [One File](https://community.obsidian.md/plugins/one-file)

Edit any markdown file on your machine from inside an Obsidian vault via symlinks, in a distraction-free focus mode.

### [Graph View Modes](https://community.obsidian.md/plugins/graph-view-modes)

Adds folder-based layout modes to Obsidian's built-in Graph View.

### [Hilo](https://community.obsidian.md/plugins/hilo)

Multi-color highlights using Obsidian's native ==text== syntax.

### Updated

### [Khiip](https://community.obsidian.md/plugins/khiip)

Capture + recall through your local Khiip daemon — URLs in, markdown in your vault, semantic recall over everything

### [SourceDown](https://community.obsidian.md/plugins/sourcedown)

Import documents, media, archives, and web pages as Markdown using Microsoft MarkItDown.

### [Math Plotter](https://community.obsidian.md/plugins/math-plotter)

Create math graphs from a GUI. Insert plots via forms without writing plot code.

### [Style HTML Viewer](https://community.obsidian.md/plugins/style-html-viewer)

Renders HTML documents natively in Obsidian workspace tabs with local CSS, image, and script asset resolution, CSP security, and link interception.

### [Slogs Sync](https://community.obsidian.md/plugins/slogs-sync)

Synchronize Markdown notes with a Slogs remote Obsidian vault.
